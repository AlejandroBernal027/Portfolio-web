# Portfolio

Esta aplicación es para que sea más accesible y sencillo poder mirar que proyectos he realizado.

Los proyectos que están en el portfolio son:

* IndieCove, mi trabajo de fin de curso.
* Drinks & Dices, una maquetación de como sería un proyecto de empresa que hice para la asignatura Empresa e Iniciativa Empresarial.
* Portfolio, la propia aplicación.
